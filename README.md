# 223nTech

## このリポジトリについて

このリポジトリは、223n.techのドメイン管理やブログ管理とかをするためのリポジトリです。

## 必要な設定など

環境変数に下表の値を設定する必要があります。

| 変数名            | 設定する値                             |
| ----------------- | -------------------------------------- |
| CLOUDFLARE_EMAIL  | Clougflareに登録しているメールアドレス |
| CLOUDFLARE_TOKEN  | Global API Key                         |
